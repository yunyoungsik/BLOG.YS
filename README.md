## 설치
```
npm install sass 
npm install next-auth
npm install prisma
npm install @prisma/client
npm install @auth/prisma-adapter
npm install moment
```
```
npx prisma init --datasource-provider mongodb   //prisma 파일 생성
npx prisma generate                             //prisma 연결
npx prisma studio                               //prisma 화면
```
